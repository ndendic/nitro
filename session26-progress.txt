@action Decorator Usage:

Basic:
```python
class Counter(Entity):
    @action()
    async def increment(self):
        self.count += 1
        self.save()
```

Advanced:
```python
@action(
    method="POST",
    path="/custom/path",
    response_model=CounterResponse,
    status_code=201,
    tags=["counters"],
    summary="Increment counter",
    description="Increments the counter by a specified amount"
)
async def increment(self, amount: int = 1):
    ...
```

URL Generation:
- Counter.increment() â†’ POST /counter/{id}/increment
- Product.create() â†’ POST /product/create (class method, no {id})
- Order.place() â†’ POST /order/{id}/place

Convenience Decorators:
```python
@get()           # GET request
@post()          # POST request (default status 200)
@put()           # PUT request
@delete()        # DELETE request (default status 204)
```
